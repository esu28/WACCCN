\subsection{Wavelet Transforms: Formal Definitions}
\label{supp:wavelet-math}

This section records the formal matrix expressions underlying the real $M$-band DWT used throughout WACCCN. Let $X\in\mathbb{R}^{n\times n}$ denote a single-channel input (gene field or histology channel). When $n$ is not divisible by $M$, we apply symmetric padding to preserve structural continuity while avoiding artifacts from zero or circular padding. After transforming, we crop padding back to the native support~\cite{Chaux2006}.

A 1D $M$-band transform is encoded by
\[
W \;=\;
\begin{bmatrix}
W_\alpha \\ W_{\beta^{(1)}} \\ \vdots \\ W_{\beta^{(M-1)}}
\end{bmatrix}
\in\mathbb{R}^{M(n/M)\times n},
\]
where each $W_f$ corresponds to convolution with filter $f$ followed by downsampling by $M$. In 2D, the level-1 subbands are
\[
S^{(1)}_{f,g} = W_f\, X\, W_g^\top ,\quad f,g\in\{\alpha,\beta^{(1)},\dots,\beta^{(M-1)}\},
\]
with $S^{(1)}_{\alpha,\alpha}$ the approximation band and the remaining $M^2{-}1$ bands capturing oriented detail. Deeper levels recursively apply the same construction to the LL band:
\[
S^{(\ell+1)}_{f,g}=W_f\,S^{(\ell)}_{\alpha,\alpha}\,W_g^\top .
\]
For histology, each of the RGB channels is padded to dimensions divisible by $M^R$, transformed for $R$ levels, and cropped back to the Visium field of view.

\subsection{Marker Detection}
\label{supp:markers}

Marker genes for visualization and embeddings are identified using a Wilcoxon rank-sum test with Benjamini--Hochberg correction. For groups $A,B$ and expression values $x_{ig}$,
\[
\bar{x}_{A,g}=\tfrac{1}{n_A}\sum_{i\in A}x_{ig},\qquad
\bar{x}_{B,g}=\tfrac{1}{n_B}\sum_{i\in B}x_{ig},
\]
\[
\mathrm{log2FC}_g = \log_2\!\left(\frac{\bar{x}_{A,g}+\varepsilon}{\bar{x}_{B,g}+\varepsilon}\right).
\]
Genes with FDR $<0.05$ and $\log_2\mathrm{FC}>1$ are retained.

\subsection{Mask-aware Neighborhood Averages}
\label{supp:mask-average}

For fields defined on the histology grid we compute masked, weighted means. Let $M$ be the binary tissue mask and $\Omega_q=\{p:\|p-q\|_2\le r\}$. Gaussian weights
\[
w_{q,p}=\exp\!\left(-\tfrac{\|p-q\|_2^2}{2\sigma_w^2}\right)
\]
are applied only to valid tissue pixels $M_p{=}1$. The masked mean is
\[
\langle X\rangle_q =
\frac{\sum_{p\in\Omega_q} w_{q,p}X_pM_p}{\sum_{p\in\Omega_q} w_{q,p}M_p+\varepsilon}.
\]
If the effective denominator is below $\varepsilon$, we return NaN.

\subsection{Anisotropy Variants}
\label{supp:aniso-variants}

We implement anisotropy using three complementary transform families: 2-band DWT, 4-band DWT, and DTCWT. Each produces subband-specific orientation metadata $(\theta_b,\kappa_b)$ used during corridor pooling.

\paragraph{2-band DWT.}
Level-1 and level-2 LH, HL, and HH maps are normalized and averaged across RGB channels. Corridor pooling aggregates these power maps to yield horizontal, vertical, and diagonal energies $(E_h,E_v,E_d)$. A reliability score
\[
r_{\mathrm{axis}}=\frac{|S_x-S_y|}{S_x+S_y+\varepsilon}
\]
with $S_x=E_v+\eta E_d$ and $S_y=E_h+\eta E_d$ provides an anisotropy index mapped to an axis ratio $a\in[1,a_{\max}]$.

\paragraph{4-band DWT.}
Fifteen oriented subbands provide rich directional information. Corridor-pooled powers $\bar P_b$ and reliability weights $\kappa_b$ define a doubled-angle resultant
\[
m=\sum_{b:\,\kappa_b\ge\tau_\kappa} \kappa_b\,\bar P_b\,e^{i2\theta_b},
\]
yielding $\theta^\star=\tfrac12\arg(m)$ and normalized anisotropy $r=|m|/(\sum\kappa_b\bar P_b+\varepsilon)$.

\paragraph{Dual-tree Complex Wavelets.}
Six oriented complex subbands at angles $\{\pm15^\circ,\pm45^\circ,\pm75^\circ\}$ provide orientation-sensitive histology descriptors. Corridor pooling collects magnitudes and phase coherence values. A weighted resultant
\[
m=\sum_{\ell,b}\omega^{(\ell)}_b\,\bar P^{(\ell)}_b\,e^{i2\theta_b}
\]
gives principal axis $\theta^\star$ and anisotropy strength.

\paragraph{Kernel Construction.}
All variants map $r$ to an axis ratio
\[
a = 1+(a_{\max}-1)r^\gamma,
\]
then form an elliptical Gaussian kernel using variances $(\sigma_\parallel^2,\sigma_\perp^2)=(1,1/a)$ with a short-axis floor for stability. Detailed hyperparameters appear in Supplementary Section~\ref{sec:aniso-hparams}.

\subsection{CCC Hyperparameters}
\label{sec:tuning_ccc}

CCC inference uses winsorized ($p99$) and min--max scaled expression. Secreted ligand dispersion uses a combined anisotropic + radial kernel; membrane-bound interactions use a near-contact radius. Receptor engagement uses a Hill function with $K_{\mathrm{half}}{=}0.01$. Null distributions are generated with $n_{\mathrm{null}}{=}1000$ matched-marginal surrogates. The morphology gate $G_{i\to j}$ is derived from low-level histology features and reused identically across all nulls.

\subsection{Autoencoder Objective}
\label{supp:autoencoder-details}

Let $Z$ be latent embeddings and $X^{(v)}$ the $v$-th input view. The multi-term objective is:

**Reconstruction.**
\[
\mathcal{L}_{\text{rec}}
=\sum_{v}\frac{1}{F_v}\cdot\frac{1}{N}\sum_i\!\left\lVert X^{(v)}_{i\cdot}-\hat X^{(v)}_{i\cdot}\right\rVert_2^2.
\]

**CCC-Laplacian Smoothness.**
Let $W$ be the row-normalized average CCC affinity:
\[
\mathcal{L}_{\text{lap}}
=\tfrac{1}{N}\sum_i\!\left\lVert Z_{i\cdot}-\sum_j W_{ij} Z_{j\cdot}\right\rVert_2^2.
\]

**Contrastive (InfoNCE).**
Using cosine similarity and a temperature $\tau=0.12$:
\[
\mathcal{L}_{\text{InfoNCE}}
=\frac{1}{N}\sum_i\frac{1}{|\mathcal{P}(i)|}\sum_{j\in\mathcal{P}(i)}
\left[-\,\mathrm{sim}_{ij}+\log\!\sum_{\ell\neq i}\exp(\mathrm{sim}_{i\ell})\right].
\]

**Deep Embedded Clustering.**
\[
q_{ik}\propto (1+\|Z_i-c_k\|_2^2)^{-1},\qquad
p_{ik}=\frac{q_{ik}^2}{\sum_i q_{ik}},
\]
\[
\mathcal{L}_{\text{DEC}}
=\tfrac{1}{N}\sum_{i,k} p_{ik}\log\frac{p_{ik}}{q_{ik}}.
\]

**Total Loss.**
\[
\mathcal{L}
=\lambda_{\text{rec}}\mathcal{L}_{\text{rec}}
+\lambda_{\text{lap}}\mathcal{L}_{\text{lap}}
+\lambda_{\text{nce}}\mathcal{L}_{\text{InfoNCE}}
+\lambda_{\text{dec}}\mathcal{L}_{\text{DEC}}.
\]

\subsection{Anisotropy Metrics}
\label{supp:metrics-aniso}

All orientations are axial ($\theta\equiv\theta{+}\pi$). The spot scale $h$ is the median 1-NN distance.

**Outgoing Angular Concentration.**
\[
R_w(i)=
\frac{\left|\sum_j w_{ij}\,e^{2i\phi_{ij}}\right|}
{\sum_j w_{ij}}\in[0,1].
\]

**Net CCC direction.**
\[
\mathbf{v}_i=\sum_j w_{ij}e^{i\phi_{ij}},\quad
\alpha_i=\arg(\mathbf{v}_i),\quad
R_{\mathrm{CCC}}(i)=\frac{|\mathbf{v}_i|}{\sum_j w_{ij}}.
\]

**Global axial alignment.**
\[
A=\frac{1}{S}\sum_i \cos\!\left(2(\alpha_i-\theta_i)\right).
\]

**Alongâ€“Across Index.**
\[
\mathrm{AI}_i=\cos\!\left(2(\alpha_i-\theta_i)\right).
\]

**Neighborhood Coherence.**
\[
A_{\text{neigh}}(i)=\frac{1}{|\mathcal{N}_r(i)|}\sum_{k\in\mathcal{N}_r(i)}
\cos\!\left(2(\alpha_k-\theta_k)\right).
\]

**Orientation Variogram and Coherence Length.**
\[
V(\delta)
=\mathbb{E}\!\left[\cos\!\bigl(2(\theta(x{+}\delta)-\theta(x))\bigr)\right],\quad
L_c=\inf\{\delta: V(\delta)\le 0.2\}.
\]

\subsection{Niche Agreement Metrics}
\label{supp:metrics-niche}

We report ARI, NMI, V-measure, one-to-one accuracy, Purity, macro/weighted F1, and silhouette scores. ARI is:
\[
\mathrm{ARI}
=\frac{\sum_{ij}\binom{n_{ij}}{2}
-\frac{\sum_i \binom{a_i}{2}\sum_j \binom{b_j}{2}}{\binom{n}{2}}}
{\tfrac12\!\left[\sum_i\binom{a_i}{2}+\sum_j\binom{b_j}{2}\right]
-\frac{\sum_i \binom{a_i}{2}\sum_j \binom{b_j}{2}}{\binom{n}{2}}}.
\]
